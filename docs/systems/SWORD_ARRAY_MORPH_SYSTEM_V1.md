# 剑阵形变系统 V1 设计方案

## 1. 设计目标

当前“冻结弹幕 -> 吸收 -> 发射”系统已经具备爽感，但主要爽点仍然来自数量倾泻。  
它的问题不是不爽，而是“强度表达不够有层次”，因此在整体体验上还不如御剑状态那样具有强烈的控制感和主导感。

本方案的目标是：

- 不增加额外操作负担
- 保留玩家控制权
- 让发射冻结弹幕从“吐出资源”升级为“展开剑阵”
- 让不同发射形态体现不同战术用途
- 与项目的“御剑 / 剑修 / 以意御形”主题保持一致

## 2. 控制原则

- 不增加新按键
- 不使用滚轮切换
- 不采用自动战场判断
- 直接由“准星与玩家距离”决定剑阵形态

这意味着玩家不是在切技能，而是在操控剑势形态。

## 3. V1 形态规则

### 3.1 当前定案阈值

- `0 - 160`：环阵
- `160 - 420`：扇阵
- `420+`：贯穿阵

### 3.2 环阵

- 触发：准星贴近玩家
- 定位：解围、护身、近身清理
- 当前实现：剑丸按环绕玩家的方向依次发射

### 3.3 扇阵

- 触发：准星处于中距离
- 定位：中距离主力形态
- 当前实现：剑丸在扇角内分布发射
- 当前要求：三种形态里面积最小，更集中、更精准

### 3.4 贯穿阵

- 触发：准星拉远
- 定位：打远端威胁、打 Boss、打线性路径
- 当前实现：剑丸沿窄直线方向连续发射

## 4. V1 预览要求

- 环阵：显示玩家周围的圆环预览
- 扇阵：显示前方扇形预览
- 贯穿阵：显示细长剑轨预览

预览规则：

- 长按发射前即可感知当前形态
- 预览必须尽量接近实际结算范围
- 优先保证可读，而不是复杂特效

## 5. V1 当前实现边界

V1 已实现：

- 三段剑阵
- 准星距离判定
- 实时预览
- 持续发射时的形态差异

V1 暂不实现：

- 连续无级形变
- 自动追踪
- 融合升阶
- 手动模式切换

## 6. 后续建议

下一阶段优先完善：

1. 发射前短暂成阵反馈
2. 三形态更强的视觉区分
3. 三形态发射节奏差异
4. 结算范围与预览进一步对齐

## 7. 后续阶段规划

以下内容不是 V1 当前定案范围，但属于剑阵系统后续明确要推进的方向。

### 7.1 V1.5：强化当前三形态表现

目标：

- 让环阵、扇阵、贯穿阵不只是“方向不同”，而是真正拥有不同的剑势感

建议内容：

1. 发射前短暂成阵
2. 三形态更强的视觉区分
3. 三形态不同的发射节奏
4. 预览与实际结算进一步对齐

#### 发射前短暂成阵

- 环阵：剑丸沿玩家周围短暂排布后再释放
- 扇阵：剑丸先在前方扇面内成列
- 贯穿阵：剑丸先收束成一条细线再贯出

目标体验：

- 从“发射弹幕”升级为“展开剑阵后释放”

#### 三形态更强的视觉区分

- 环阵更强调护体和领域感
- 扇阵更强调锐利和中距压制
- 贯穿阵更强调剑轨和破线感

#### 三形态不同的发射节奏

- 环阵：更均匀，更像持续护身爆散
- 扇阵：更稳定，更像中距主力清场
- 贯穿阵：更集中，更像短时间刺出多段剑流

#### 预览与结算对齐

- 玩家看到的区域应与实际命中区域尽量接近
- 这是后续所有升级的基础

### 7.2 V2：从“三段形态”升级到“连续形变”

目标：

- 让剑阵看起来真正是连续变形，而不是硬切三种模式

建议内容：

1. 环阵向扇阵渐开
2. 扇阵向贯穿阵渐收
3. 预览先连续变化
4. 结算是否连续化再单独评估

说明：

- V2 不建议一开始就让伤害判定完全连续变化
- 更稳的方式是先把视觉和排布做连续
- 如果手感稳定，再决定是否把实际结算也连续化

### 7.3 V2.5：加入形态专属战术价值

目标：

- 让三种形态的差异不只体现在覆盖方式上，也体现在机制价值上

建议方向：

#### 环阵

- 更强的近身清弹
- 近身敌人命中后轻微击退
- 在高压环境下提供更明确的解围感

#### 扇阵

- 更稳定命中多个普通敌人
- 更适合处理中距离散怪和半包围局面
- 维持作为“默认主力形态”的地位

#### 贯穿阵

- 更适合线性穿透
- 更适合打 Boss 与远距离高价值目标
- 强化“破阵 / 穿刺 / 清路径”定位

### 7.4 V3：引入剑丸融合与高阶剑阵

目标：

- 在不破坏当前控制逻辑的前提下，把资源积累做出更强层次

建议内容：

1. 吸收越多，剑丸可融合升阶
2. 数量减少，但质量提升
3. 同一种剑阵形态在高阶时拥有更强视觉和更高强度

示例方向：

- 环阵升级为双层护身阵
- 扇阵升级为多段扇面剑雨
- 贯穿阵升级为重型剑柱或多段穿刺

说明：

- 融合应在基础形态稳定后再做
- 否则系统会同时引入“形态控制”和“资源升阶”两层复杂度，风险太高

### 7.5 长期目标：让剑阵系统成为与御剑并列的第二核心

最终目标不是让剑阵系统变成“另一个输出键”，而是让它成长为：

- 可操控空间结构的战斗系统
- 与御剑时停并列的核心识别点
- 负责处理中近距离场面与资源释放高潮的主机制

也就是说：

- 御剑负责时间与机制控制
- 剑阵负责空间与资源控制

两者互补，而不是互相替代。
