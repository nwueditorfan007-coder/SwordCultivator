# 第一章：入阵 波次引导方案

## 文档定位

本文档记录第一章教学关卡的波次设计与实现依据。

目标不是通过文字教程讲完所有系统，而是用敌人组合逐步训练玩家的身体判断：

`近战稳住局面 -> 御剑处理关键点 -> 剑阵按距离兑现输出 -> Boss 切丝开窗`

本方案优先服务 `res://scenes/main/Main.tscn` 当前主玩法。若后续强化剑技 / 剑丸窗口正式落地，本章可以作为“常规战斗基础课”，再在第二章引入强化窗口。

## 设计原则

- 每一波只新增一个主要战斗问题。
- 敌人不是技能锁，而是问题来源；正确技能应更顺、更稳、更爽，但不应成为唯一解。
- 先建立按钮可信度，再增加战场判断。
- 近战、御剑、剑阵三者按输入复杂度递进。
- 每个解锁波都应给玩家约 `9-10` 次自然使用新动作的机会，再进入下一课。
- Boss 首战只检查核心循环，不追求完整状态复杂度。

## 玩家应被训练出的判断

第一章结束时，玩家不需要能复述系统名，但应自然形成以下判断：

- 贴身压力来了，先用近战稳住。
- 远处高价值目标很烦，用御剑处理。
- 连线 / 丝线是机制目标，优先切断。
- 近距离用环阵解压，中距离用扇阵扫面，远距离用贯穿阵破线。
- Boss 开窗后再集中兑现输出。

## 阶段结构

### 解锁线

本章采用逐步解锁，先给玩家建立输入安全感，再逐步增加战场判断：

- 第 `1` 波：解锁近战。
- 第 `2` 波：解锁御剑。
- 第 `3` 波：解锁剑阵 `环阵`。
- 第 `4` 波：解锁剑阵 `扇阵`。
- 第 `5` 波：解锁剑阵 `贯穿阵`。

第 `1-5` 波不是“一次展示题”，而是“单技能练习波”。每波通过 3-4 个小段落重复同一个战斗命题，让玩家把动作熟悉到能自然做出来，再解锁下一步。

第 `6` 波开始不再新增基础输入，而是把已经学到的技能组合成更典型的战斗命题。

### 阶段 1：近战课

目标：让左键短按成为玩家的安全按钮。

玩家体感应是：

- 我能靠移动和近战处理第一批敌人。
- 敌弹不是纯惩罚，弹反后可以变成反击机会。
- 贴身危险不是失控，而是提醒我回到近战节奏。

### 阶段 2：御剑课

目标：让右键成为处理高价值目标和制造时间差的工具。

玩家体感应是：

- 远处目标靠近战处理很累。
- 右键点刺 / 连斩能让我主动处理关键点。
- 子弹时间不是装饰，而是给我争夺战场处理权。

### 阶段 3：剑阵三形态课

目标：用敌人站位训练环 / 扇 / 贯的距离语义。

玩家体感应是：

- 近身密集压力适合环阵。
- 中距离散布敌人适合扇阵。
- 远距离关键目标或线性站位适合贯穿阵。

### 阶段 4：切线课

目标：在 Boss 前预教学 Silk 逻辑。

玩家体感应是：

- 连线存在时，局面会变麻烦。
- 切断线后，压力立刻下降。
- 线是机制目标，不是背景装饰。

### 阶段 5：Boss 考试

目标：检查完整核心循环。

玩家体感应是：

- Puppet 不是普通输出对象。
- Silk 才是机制处理目标。
- 切完 Silk 后 Boss 开窗，才进入输出兑现。
- 紫色 Boss 是闭窗期，黄色 Boss 才是可输出窗口。

## 典型化实现要求

第一版典型化不只改敌人数量，还要改刷怪语法：

- 脚本化波次允许指定相对玩家的出生偏移。
- 同一波可以用 `delay_after` 分段刷怪，形成“展示题 -> 小应用”的节奏。
- 教学波优先从玩家正前方或清晰扇面 / 纵线出生，减少随机边缘刷怪稀释教学意图。
- 第 `1-5` 波必须保持足够练习密度：不要让玩家只做 2-3 次新动作就解锁下一项能力。
- 解锁前允许飞剑本体可见，但未解锁技能输入必须给即时反馈。
- 剑阵形态按最高已解锁形态限制：环阵前不能开剑阵；扇阵前只保留环阵；贯穿阵前不进入贯穿语义。
- `距离控阵` 和 `Space切阵` 都必须遵守同一条解锁线：手动切阵只在当前已解锁形态中循环。

## Boss 首战约束

第一章 Boss 不应急着展示完整难度。

首战建议只强化三段：

1. 观察段：低压 `Thousand Silks`，让玩家读 Boss 位置和弹幕节奏。
2. 机制段：`Puppet Ambush` 生成少量 Puppet / Silk，要求御剑切 Silk。
3. 破绽段：清空 Silk 后打开短破绽，鼓励近战或剑阵输出。

暂不强调：

- 高密度 `Silk Cage`
- 高频 `Needle Return`
- 第二只 Boss
- 纯血量 / 纯伤害压迫

## 第一版实现范围

第一版先实现以下内容：

- 前 9 波使用手工队列与典型构图。
- 第 10 波进入 Boss。
- 第 10 波后回到通用随机波次。
- 第 1-5 波逐步解锁近战、御剑、环阵、扇阵、贯穿阵。
- 第 1-5 波加密为 3-4 段练习题，让玩家约完成 9-10 次对应动作后再进入下一课。
- 不新增敌人类型。
- 不修改敌人 AI 核心行为。
- 不增加新 UI 教程。

## 验证问题

下一轮试玩重点观察：

1. 玩家是否在第 1 波把近战当作安全按钮。
2. 玩家是否在第 2 波自然尝试右键处理远处目标。
3. 玩家是否在第 3-5 波能按解锁顺序分辨环 / 扇 / 贯的收益。
4. 玩家是否在第 6 波把连线读成机制目标。
5. 第 10 波 Boss 是否让玩家完成“切 Silk -> 开窗 -> 输出”的闭环。
