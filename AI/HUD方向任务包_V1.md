# 剑修 Demo HUD 方向任务包 V1

## 1. 任务目的

本任务包用于产出 `3` 套 HUD 方向稿。

目标不是立刻定终版，而是先把“信息结构”升级成“法器式 HUD 语言”。

三套方向稿都必须满足：

- 仍然服务战斗
- 不遮挡中心战场
- 有东方修仙法器感
- 与主视觉母版兼容


## 2. 共有信息结构

三套 HUD 都必须包含：

- 左上：生命、剑意、波次、得分、飞剑数量
- 顶中：标题或状态短语
- 右上：模式 / 状态徽记
- 左右侧：题字或箴言装饰
- 底部：技能盘、按键提示、操作说明

不新增玩法信息，不改变战斗逻辑。


## 3. 共通风格要求

风格关键词：

- 古金细线
- 法器边框
- 阵图嵌件
- 冷白青蓝能量
- 深蓝黑底
- 轻薄
- 克制

硬性要求：

1. UI 看起来像法器构件，而不是现代面板。
2. 资源条必须有嵌入阵图的感觉。
3. 技能盘要像符印或阵眼，不要像 app 图标。
4. 中央战斗区必须尽量让开。
5. 小字数值必须清楚可读。

禁止项：

- 厚重金属边框
- 卡牌式面板
- 大块实体按钮
- 科幻仪表盘
- 大面积发光霓虹


## 4. 方向 A：极简天仪版

### 定位

- 最克制
- 最接近战斗读感优先
- 适合作为最稳的首版落地方向

### 结构

- 左上仅保留简洁图腾 + 两条细资源条
- 顶部中央仅保留一行题字和极细边饰
- 右上为单个圆形模式徽记
- 底部技能区以轻量圆印排列
- 两侧竖排文字很轻，只做氛围

### 提示词

```text
为一款中国修仙题材的俯视角动作游戏设计一套极简 HUD。整体像一件法器的铭纹而不是现代面板。左上资源区由小型图腾、极细生命条和剑意条构成，顶部中央只有一行题字和微弱边饰，右上只有单个圆形模式徽记，底部是一排轻量技能圆印和按键提示。整体以深蓝黑、古金细线、冷白青蓝能量为主，极其克制，留白很多，中心战场尽量让开。
```

### 验收点

- 一眼清爽
- 战斗区最大
- UI 最不抢戏


## 5. 方向 B：平衡法器版

### 定位

- 推荐主推方向
- 气质、可读性、完成度最平衡
- 最适合作为 demo 首版 UI

### 结构

- 左上资源区带莲花 / 阵图图腾
- 资源条嵌在法器结构中
- 顶部中央题字与边饰更完整
- 右上模式徽记更像金色法印
- 底部技能盘具备较清晰的主次关系
- 左右侧题字与角饰存在感适中

### 提示词

```text
为一款中国修仙题材的俯视角动作游戏设计一套平衡型 HUD。整体像一件古老法器的边框和阵图结构，既高级克制，又有明确完成度。左上资源区包含图腾、生命条、剑意条、波次与飞剑信息，资源条嵌在古金细线结构内；顶部中央有较完整的题字和边饰；右上是法印式模式徽记；底部技能盘有主次关系清晰的技能圆印和按键提示；左右侧带轻度题字与角饰。风格为深蓝黑底、古金秩序线、冷白青蓝能量、微弱发光、东方神秘感、法器感。
```

### 验收点

- 信息清楚
- 气质成立
- 不显廉价


## 6. 方向 C：华丽星图版

### 定位

- 最接近参考图上限
- 用于探索上限密度
- 主要做审美边界测试

### 结构

- 左上资源区更完整
- 顶部中央更有仪式感
- 底部技能盘更像阵盘
- 两侧文字与云纹存在感更强
- 允许更多星图和边框细节

### 提示词

```text
为一款中国修仙题材的俯视角动作游戏设计一套偏华丽的星图式 HUD。整体像天仪、阵盘和法器融合而成，具有玄夜星图感和东方仪式感。左上资源区更完整，包含图腾、资源条、波次和飞剑信息；顶部中央题字区域更有仪式感；右上模式徽记更复杂；底部技能区像小型阵盘；左右侧带书法题字、云纹和金线边饰。风格依旧克制高级，不做页游堆料，中心战场必须保留明确留白。
```

### 验收点

- 华丽但不脏
- 仍然能打
- 不挤压战斗区


## 7. 统一评审问题

三套方向稿都要回答以下问题：

1. 哪一套最不影响战斗读感？
2. 哪一套最像“法器”，而不是“UI 皮肤”？
3. 哪一套最适合当前 demo 的资源投入？
4. 哪一套最容易与后续背景和飞剑特效融合？
5. 哪一套最有机会继续升级，而不是一开始就走死？


## 8. 推荐筛选方法

第一轮先不看“哪套最炫”，先筛：

- 是否清楚
- 是否统一
- 是否能进战斗

如果一套 HUD 很华丽，但把中心战斗压碎，直接淘汰。


## 9. 二轮修订指令

### 只压密度

```text
保持 HUD 整体风格与布局不变，仅减少装饰件数量和亮度，让信息更清晰。
```

### 只提法器感

```text
保持布局不变，仅加强法器结构感和阵图嵌入感，减少现代 UI 面板既视感。
```

### 只提可读性

```text
保持风格不变，仅强化数值、标签和技能区主次关系，让 HUD 在战斗中更清楚。
```


## 10. 当前推荐结论

当前阶段最有可能最终落地的是：

- 以 `方向 B：平衡法器版` 为主
- 用 `方向 A：极简天仪版` 保底
- 用 `方向 C：华丽星图版` 提供审美上限参考

也就是说：

- `A` 负责保战斗
- `B` 负责进产品
- `C` 负责抬天花板
